Here's a small example of the OAuth library for JavaScript in action. You can fill out a form to request a token. Then copy and paste the token into the form to authorize your consumer. Then copy and paste the token and token secret into the form to get an access token.

When using Microsoft Internet Explorer, you can prevent annoying warnings by checking the box 'Allow active content to run in files on my computer' in 'Tools > Internet Options ... > Advanced > Security'.
